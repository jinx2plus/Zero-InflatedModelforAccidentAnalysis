# Research Summary

## 1) 목적
졸음쉼터 설치 전/후 사고 건수 데이터에서 사고 발생에 영향을 주는 요인을 식별하고,
설치 전후 효과(`Is_After`)의 상대적 중요도를 확인한다.

## 2) 방법
- 모델: Zero-Inflated Negative Binomial (ZINB) Neural Network
- 타깃: `ACCIDENTS_BEFOREINSTALLATION_VERSION*`, `ACCIDENTS_AFTERINSTALLATION_VERSION*`
- 데이터 구성: 동일 구간의 전/후 데이터를 `Is_After` 플래그(0/1)로 스택
- 학습/평가: Train/Test split, ZINB negative log-likelihood 최소화
- 변수 중요도: Permutation Importance (손실 증가량)

## 4) 해석 포인트
- 두 버전 공통으로 속도/연속주행/도로구조(차로수, 톨게이트, 곡선반경) 변수가 상위권을 차지한다.
- `Is_After` 중요도는 낮거나 음수(Version1: -0.13, Version2: -0.05)로 나타났다.
  - 설치 전후 구분 자체보다 교통/도로 특성 변수가 모델 손실에 더 직접적으로 기여했음을 시사한다.
- 본 중요도는 예측 관점의 상대 기여도이며, 인과효과를 직접 증명하지는 않는다.

## 5) 재현 시 주의사항
- 원본 데이터는 공개 저장소에 포함하지 않는다.
- 데이터 파일 경로는 `--data-path` 또는 `DATA_FILE` 환경변수로 전달한다.
- 결과 CSV는 `results/`에 저장한다.
